# Changelog

All notable changes to TaskFlow will be documented in this file.

## [2.3.0] - 2026-06-20

### Added
- Loading screen with animated logo
- Voice input processing indicator with spinner
- Custom focus duration input (1-180 minutes)
- Subtask progress bars
- Achievement persistence across sessions

### Fixed
- Voice input false date detection
- Voice input premature stop handling
- Improved error messages for voice input
- Better transcript processing

## [2.2.0] - 2026-06-20

### Added
- Manual voice input toggle
- Custom focus duration selection
- Subtask progress visualization

## [2.1.0] - 2026-06-20

### Added
- Inline task editing
- Swipe gestures (mobile)
- Subtasks within tasks
- Haptic feedback

## [2.0.0] - 2026-06-20

### Added
- Complete UI rewrite
- Premium dark theme
- Focus mode with Pomodoro timer
- Task templates
- Statistics and achievements
- Voice input
- Trash/recycle functionality

## [1.0.0] - 2026-06-13

### Added
- Initial release
- Basic task management
- Categories and priorities
- Due dates
- Dark mode
- Local storage persistence
